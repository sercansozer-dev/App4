GoPxL .NET SDK Documentation

API Reference

For detailed API documentation, see the API Reference section.

Overview

The GoPxL .NET SDK is a C# API library that supports communication with GoPxL sensors and systems.

The GoPxL .NET SDK is an open-source library provided under the MIT license.

Sample Applications

The SDK includes 17 sample applications demonstrating various SDK features:

  • Sensor configuration and control
  • Data reception (profiles, surfaces, measurements, images)
  • Multi-sensor layouts and alignment
  • Job management and replay functionality
  • Network discovery

Sample source code is located in the samples/ directory.

Building the SDK

A generate.py script is included in the SDK that generates a Visual Studio solution file for the .NET SDK library and all sample applications.

Prerequisites

  • Python 3.7+ (for running generate.py)
  • Visual Studio 2017 or later
  • .NET Framework 4.7.2 or later

Generating and Building

Run the generate script from the GoPxLSdkNet directory:

> cd *GoPxL_SDK/Gocator/GoPxLSdkNet
> python -m generate

This will create a Visual Studio solution file (GoPxLSdkNet.sln) that includes the SDK library and all sample projects. Open the solution in Visual Studio 2017 or later and build.

Platform Support

Note: The .NET SDK and samples are Windows-only. For cross-platform C++ SDK support, see the GoPxLSdk directory.

Tested Operating Systems

  • Windows 10 (64-bit)

Compiler Requirements

  • Visual Studio 2017 or later

Units

The GoPxL .NET SDK uses mm, mm², mm³, hertz, microseconds, and degrees as standard units unless otherwise mentioned.
